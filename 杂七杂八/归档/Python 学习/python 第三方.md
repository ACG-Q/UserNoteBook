### python 第三方模块click

>  click模块 可以 轻松将一个函数变成一个命令行工具
>
> ##  [文档](http://click.pocoo.org/6/)
>
> ### 搭配 argparse 来使用
>
> > 此模块是 Python 标准库中推荐的命令行解析模块。
>
> ### 效果等同于 docopt
>
> > 用了这个家伙可以在命令行执行py文件时传递参数，例如 ：python test.py --tread_num 10
